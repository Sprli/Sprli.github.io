背景

模块一：按键K1控制LED指示灯的颜色

  • 相关函数：
    主要使用了ybrgb库中的设置LED显示函数set(x, x, x)以及ybkey库中的判断按键是否按下函数is_press()

    • set(red_set, green_set, blue_set)
      • red_set表示红色的控制亮灭，其中1表示亮，0表示灭。green_set和blue_set同理
    • is_press()
      • 返回值为1表示检测到K1按键按下，返回值0表示未检测到K1按键按下

  • 具体代码：

    # Untitled - By: HP - 周日 8月 20 2023

import time, lcd, image
from modules import ybkey, ybrgb

lcd.init()
lcd.clear(lcd.WHITE)
RGB = ybrgb()
Key = ybkey()
mode = 0
while True:
    if Key.is_press():
        time.sleep_ms(20)
        while Key.is_press():
            pass
        mode += 1
        mode %= 8 
    if mode == 0:
        RGB.set(0, 0, 0)
    elif mode == 1:
        RGB.set(0, 0, 1)
    elif mode == 2:
        RGB.set(0, 1, 0)
    elif mode == 3:
        RGB.set(0, 1, 1)
    elif mode == 4:
        RGB.set(1, 0, 0)
    elif mode == 5:
        RGB.set(1, 0, 1)
    elif mode == 6:
        RGB.set(1, 1, 0)
    elif mode == 7:
        RGB.set(1, 1, 1)
    
    lcd.draw_string(140, 120, "Mode:" + str(mode), lcd.BLACK, lcd.WHITE)
代码二：使用定时器中断运行函数

  • 相关函数：
    主要使用到了Timer库中的Timer定时器配置函数

    • Timer(id, channel, mode=Timer.MODE_ONE_SHOT, period=1000, unit=Timer.UNIT_MS, callback=None, arg=None, start=True, priority=1, div=0)
      • id : Timer ID, [0~2] (Timer.TIMER0~TIMER2)
      • channel : Timer 通道, [Timer.CHANNEL0~Timer.CHANNEL3]
      • mode : Timer 模式, MODE_ONE_SHOT 或者 MODE_PERIODIC 或者 MODE_PWM
      • period : Timer 周期, 在启动定时器后 period 时间， 回调函数将会被调用，(0,~)
      • unit : 设置周期的单位，默认位毫秒（ ms ），Timer.UNIT_S 或者 Timer.UNIT_MS 或者 Timer.UNIT_US 或者 Timer.UNIT_NS
      • callback : 定时器回调函数， 定义了两个参数， 一个是定时器对象 Timer ， 第二个是在定义对象时希望传的参数 arg ，更多请看 arg 参数解释
      • arg : 希望传给回调函数的参数，作为回调函数的第二个参数
      • start : 是否在对象构建成功后立即开始定时器， True ：立即开始； False :不立即开启，需要调用 start() 函数来启动定时器
      • priority : 硬件定时器中断优先级， 与特定的CPU相关， 在K210中，取值范围是[1,7]， 值越小优先级越高
      • div : 硬件定时器分频器，取值范围[0,255]， 默认为0， clk_timer（定时器时钟频率） = clk_pll0（锁相环0频率）/2^(div+1)
    • on_timer(timer)
      • 该函数可以理解为定时器中断函数，即每隔一确定时间该函数就会执行一次

  • 具体代码：
